Goals
-----
* 기본적으로 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임
* 같은 수가 같은 자리에 있으면 스트라이크
* 다른 자리에 있으면 볼
* 같은 수가 전혀 없으면 포볼 또는 낫싱
* 힌트를 이용해서 먼저 상대방(컴퓨터)의 수를 맞추면 승리

RandomNum
------------
* 서로 다른 3개의 숫자를 컴퓨터는 준비해야 한다!!
